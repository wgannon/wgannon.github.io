About Me

I'm a GIS Tech with asperations for being a developer. I first found my love of Maps in college then later, while on the job, I began learning a bit about coding and really enjoy creating scripts to speed up workflows and processes. I create some useful tools for processing histroical plate maps and field data and create construction plans. In my free time I like to have gaming nights with my friends so you could say I'm very team oriented when attacking difficult tasks. I also have been increasly interested in the hardware behind current tech. From using my own router instead of paying for the ISP's, fixing my Phone and PS4, to building my own computer from scratch. Infact I'm looking to upgrade some of my parts after a few years of use.

Python Regression Software



This was a project ot make an application for a simple regression analysis using python and tools from ArcGIS Pro. It is meant to be a tool for a specific use, this one being an inerative tool that runs a regression and prints a pdf map of the output for it without having to go into Pro at all. You just need to adjust the K Value and outputfile name and it will run the regression with that K value. Built using a Tkinter GUI.
